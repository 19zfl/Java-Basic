# do……while 循环

#### do……while循环的语法结构：

​		do {

​				循环体；

​		} while （布尔表达式）；

#### do……while循环的执行原理：

- 执行循环体；
  - 判断布尔表达式
    - true
      - 执行循环体
        - 判断布尔表达式
          - true
            - ……
          - false
            - ……
    - false
      - 循环终止

#### do……while循环的循环次数：

1 - N 次 （至少一次）。

#### 使用do……while循环的注意事项：

do……while最后有一个分号。
